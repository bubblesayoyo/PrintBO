Privacy Policy

PrintBO · Last updated April 22, 2026

PrintBO is an internal business tool developed exclusively for use by Cleveland Printwear. This privacy policy describes how the application handles data.

Who Uses This App

PrintBO is not a public application. It is used solely by authorized employees of Cleveland Printwear to manage invoicing between Printavo and QuickBooks Online.

Data Accessed

  • QuickBooks Online customer records, invoice data, and account information via the Intuit QuickBooks API
  • Printavo order, quote, and customer data via the Printavo API

Data Storage

All data is stored locally on the authorized device. API credentials are stored securely in the macOS Keychain. No data is transmitted to any server other than the official Intuit QuickBooks and Printavo APIs.

Data Sharing

No data accessed by PrintBO is shared with any third parties. The application communicates exclusively with Intuit QuickBooks Online and Printavo on behalf of the user.

Data Retention

Invoice history and customer mappings are stored in a local SQLite database on the device. This data can be cleared at any time from within the application settings.

Security

PrintBO requires device authentication (Touch ID or password) to launch. API tokens and credentials are stored in the macOS Keychain and are never stored in plain text.

Contact

For questions about this privacy policy, contact Cleveland Printwear at mattcannon@clevelandprintwear.com.
